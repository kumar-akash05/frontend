Option A — Web3Forms (2 minutes, recommended)

  1. Open web3forms.com
  2. Enter kumarakash030528@gmail.com and create your Access Key
  3. Copy server/.env.example to server/.env if you have not already
  4. Paste the key: WEB3FORMS_ACCESS_KEY=your-key-here
  5. Restart the server: cd server && npm start
  6. Submit a test on the contact form — check Gmail inbox and spam

Option B — Gmail SMTP

  1. Google Account → Security → 2-Step Verification → App passwords
  2. Create an app password for “Mail”
  3. In server/.env set SMTP_USER, SMTP_PASS (see .env.example)
  4. Restart npm start in the server folder

Until email is configured, every submission is still saved — open admin-leads.html.
