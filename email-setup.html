Enable email delivery (Gmail + Nodemailer)

Leads are sent from the Node.js backend to kumarakash030528@gmail.com. The contact form shows success only after both admin and customer emails are sent.

Step 1 — Gmail App Password

  1. Open Google Account Security
  2. Turn on 2-Step Verification
  3. Search App passwords → create one for “Mail”
  4. Copy the 16-character password

Step 2 — Save password (easiest on Windows)

In PowerShell:

cd server
.\setup-email.ps1

It will ask for your Gmail App Password and save server/.env automatically.

Or edit server/.env manually — line must look like this (no spaces after =):

EMAIL_PASS=abcdefghijklmnop

Your file currently has EMAIL_PASS empty if the red banner still shows.

Step 3 — Start server

cd server
npm install
npm start

Open http://localhost:3001 and submit the form. Check terminal for [finbiz-mail] Gmail SMTP ready.
